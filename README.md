# Fake-Headline-Generator
A simple Python-based project that generates random “breaking news” headlines by combining subjects, actions, and locations. The program runs in a loop and allows users to generate multiple headlines interactively.

🚀 Features

* Generates random and unique headlines each time
* Uses Python’s random module
* Interactive user input (yes/no system)
* Clean and simple logic using loops and lists

🛠️ Technologies Used

* Python
* Random module
* Basic control structures (loops, conditions)

💡 How It Works

The program stores predefined lists of:

* Subjects
* Actions
* Places
